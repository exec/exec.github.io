Client-side developer utilities

Overview

DevTools is a collection of client-side utilities for developers. All operations happen locally in your browser - no data is sent to any server.

Available Tools

Unicode Converter

Transform text into 38+ different Unicode styles including:

  • Mathematical variants (bold, italic, script, fraktur)
  • Styled text (circled, squared, parenthesized)
  • Special effects (inverted, reversed, strikethrough)
  • Zalgo text with adjustable intensity

Hash Generator

Generate cryptographic hashes using the Web Crypto API:

  • SHA-1: 160-bit hash (legacy, not recommended for security)
  • SHA-256: 256-bit hash (most common, recommended)
  • SHA-384: 384-bit hash (extended security)
  • SHA-512: 512-bit hash (maximum security)

Encoding & Text Utilities

Comprehensive text encoding, decoding, and manipulation:

  • Encoding: Base64, URL, HTML entities, Hex, Binary
  • Text Tools: Case conversion, word count, line sorting, duplicate removal

Features

Getting Started

Quick start guide

Using DevTools

Simply navigate to any tab and start typing. All conversions happen in real-time.

Unicode Converter

  1. Click the "Unicode Converter" tab
  2. Type or paste text into the input field
  3. Browse the results below
  4. Click "Copy" on any style you want to use
  5. Adjust the Zalgo intensity slider for custom creepiness levels

Hash Generator

  1. Click the "Hash Generator" tab
  2. Enter the text you want to hash
  3. View SHA-1, SHA-256, SHA-384, and SHA-512 hashes instantly
  4. Click "Copy" to copy any hash to your clipboard

Encoding & Text

  1. Click the "Encoding & Text" tab
  2. Enter your text
  3. See all encoding/decoding/manipulation results
  4. Copy any result with one click

Keyboard Shortcuts

All input fields support standard keyboard shortcuts:

  • Ctrl/Cmd + A - Select all
  • Ctrl/Cmd + C - Copy
  • Ctrl/Cmd + V - Paste

Browser Compatibility

DevTools works in all modern browsers:

  • Chrome/Edge 60+
  • Firefox 60+
  • Safari 11+
  • Opera 47+

Privacy

All processing happens locally in your browser. No data is collected, stored, or transmitted to any server.

How to Add Docs

To add documentation for your projects:

Option 1: Edit HTML Directly

  1. Open index.html
  2. Find the .docs-sidebar section
  3. Add a new button: <button class="doc-link" data-doc="your-project">Your Project</button>
  4. Add a new article in .docs-content: 
    <article id="doc-your-project" class="doc-article">
  <h1>Your Project</h1>
  <p>Documentation content...</p>
</article>

Option 2: Add Markdown Support

You can add a markdown parser library like marked.js to write docs in Markdown format.

Option 3: Load from External Files

Create separate HTML or Markdown files and load them dynamically with JavaScript.

Documentation Structure

Each documentation article should include:

  • Title: Clear project name
  • Overview: Brief description
  • Installation: How to set up
  • Usage: Examples and guides
  • API Reference: If applicable
  • Contributing: How to contribute

Styling

The documentation uses the same dark theme as the rest of DevTools. All standard HTML elements are automatically styled:

  • Headings (h1-h6)
  • Paragraphs and lists
  • Code blocks and inline code
  • Links and emphasis
